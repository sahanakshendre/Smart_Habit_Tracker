# Smart Habit Tracker (Flask)

Smart Habit Tracker with calendar tracking, analytics, authentication and priority-based habit management. built using Flask (Python).

This application allows users to:
- Register and Login securely
- Add habits with categories and priorities (Low / Medium / High)
- Track daily completion
- Maintain streak count
- Filter habits by category
- View analytics based on categories
- Track progress percentage for selected dates
- View completed habits using a separate calendar section
