# Smart Habit Tracker (Flask)


Smart Habit Tracker is a web application built with Flask (Python) that helps users build and track their daily habits using priority-based management, calendar tracking, and analytics.




# Features




• Secure user registration and login  

• Add habits with categories and priorities (Low / Medium / High)  

• Track daily habit completion  

• Maintain streak counts  

• Filter habits by category  

• View analytics based on habit categories  

• Track progress percentage for selected dates  

• View completed habits on a separate calendar section  



# Installation



1\. Clone the repository:



&nbsp;  git clone https://github.com/sahanakshendre/Smart_Habit_Tracker.git



2\. Navigate to the project folder:



&nbsp;  cd Smart\_Habit\_Tracker



3\. Install required packages:



&nbsp;  pip install -r requirements.txt



4\. Run the application:



&nbsp;  python app.py



# Usage


• Open your browser and go to `http://127.0.0.1:5000/` 

• Register a new account or login  

• Add and track your habits 

• Check analytics and calendar views
